## 1. Purpose

This protocol ensures that no benchmark question from the Instrumentation Universal Knowledge (Dawes) adversarial benchmark appears — in whole or in recognizable part — in the training corpus used to fine-tune the model under test. Benchmark contamination produces false high scores and invalidates the evaluation. This is the equivalent of a student seeing the exam before they take it.

**The risk is real:** If a benchmark question, its answer key, or distinctive phrasing from either was present in training data, a model can reproduce the correct answer through memorization rather than reasoning. This does not test competence — it tests recall. The benchmark becomes worthless.

## 3. Fingerprint Generation — Pre-Training Step

Before any fine-tuning run, generate fingerprints from all Corpus B benchmark files. These fingerprints are used post-training to verify the model was NOT trained on benchmark content.

### 3.1 N-Gram Fingerprint Extraction

For each benchmark question file, extract all unique n-grams of length 8–12 words that are:
- Highly specific to instrumentation domain content
- Unlikely to appear verbatim in general I&E training text
- Present in the question text OR the answer key text

**Target phrases to fingerprint (examples of high-specificity strings):**
- Exact question stems: `"A pressure transmitter is specified with an accuracy of ±0.075% of span"`
- Answer key formulas: `"Stability drift = 0.1% URL/year × 250 inH₂O × 1.5 years"`
- Adversarial trap phrases: `"ISA-18.2 does NOT specify response times for alarm priority levels"`
- Diagnostic scenario setups: `"upstream and downstream block valves are open, equalizing valve is closed, process pressure is 450 PSI"`

### 3.2 Automated Fingerprint Extraction Script

```python
#!/usr/bin/env python3
"""
IUK Benchmark Fingerprint Extractor
Generates n-gram fingerprints from benchmark files for contamination detection.
Usage: python iuk_fingerprint.py --input <benchmark_dir> --output fingerprints.json
"""

import json
import re
import hashlib
from pathlib import Path
from collections import defaultdict


def tokenize(text: str) -> list[str]:
    """Lowercase, strip punctuation, split on whitespace."""
    text = text.lower()
    text = re.sub(r'[^\w\s\.\-\+\×\÷\%]', ' ', text)
    return text.split()


def extract_ngrams(tokens: list[str], n: int) -> list[tuple]:
    return [tuple(tokens[i:i+n]) for i in range(len(tokens) - n + 1)]


def fingerprint_file(filepath: Path, ngram_sizes=(8, 10, 12)) -> dict:
    """Extract fingerprints from a single benchmark file."""
    text = filepath.read_text(encoding='utf-8')
    tokens = tokenize(text)
    
    fingerprints = {}
    for n in ngram_sizes:
        ngrams = extract_ngrams(tokens, n)
        for gram in ngrams:
            gram_str = ' '.join(gram)
            gram_hash = hashlib.sha256(gram_str.encode()).hexdigest()[:16]
            fingerprints[gram_hash] = {
                'ngram': gram_str,
                'n': n,
                'source_file': str(filepath.name)
            }
    
    return fingerprints


def extract_all_fingerprints(benchmark_dir: str, output_path: str):
    """Extract fingerprints from all IUK benchmark files."""
    benchmark_path = Path(benchmark_dir)
    all_fingerprints = {}
    file_count = 0
    
    # Target all IUK benchmark markdown files
    for filepath in benchmark_path.glob('IUK_*.md'):
        print(f"  Fingerprinting: {filepath.name}")
        file_fps = fingerprint_file(filepath)
        all_fingerprints.update(file_fps)
        file_count += 1
    
    output = {
        'version': '1.0',
        'source_files': file_count,
        'total_fingerprints': len(all_fingerprints),
        'fingerprints': all_fingerprints
    }
    
    with open(output_path, 'w') as f:
        json.dump(output, f, indent=2)
    
    print(f"\nDone. {len(all_fingerprints)} fingerprints from {file_count} files → {output_path}")
    return output


if __name__ == '__main__':
    import argparse
    parser = argparse.ArgumentParser()
    parser.add_argument('--input', required=True, help='Directory containing IUK benchmark files')
    parser.add_argument('--output', default='iuk_fingerprints.json', help='Output fingerprint file')
    args = parser.parse_args()
    extract_all_fingerprints(args.input, args.output)
```

## 4. Training Corpus Scan — Pre-Training Verification

Before any fine-tuning run begins, scan the entire training corpus for fingerprint matches.

### 4.1 Corpus Scanner Script

```python
#!/usr/bin/env python3
"""
IUK Training Corpus Contamination Scanner
Scans training data files for benchmark fingerprint matches.
Usage: python iuk_scan.py --fingerprints fingerprints.json --corpus <training_dir> --report scan_report.txt
"""

import json
import re
import hashlib
from pathlib import Path


def tokenize(text: str) -> list[str]:
    text = text.lower()
    text = re.sub(r'[^\w\s\.\-\+\×\÷\%]', ' ', text)
    return text.split()


def scan_file(filepath: Path, fingerprint_db: dict, ngram_sizes=(8, 10, 12)) -> list[dict]:
    """Scan a single training file for fingerprint matches."""
    try:
        text = filepath.read_text(encoding='utf-8', errors='ignore')
    except Exception as e:
        return [{'error': str(e), 'file': str(filepath)}]
    
    tokens = tokenize(text)
    matches = []
    
    for n in ngram_sizes:
        for i in range(len(tokens) - n + 1):
            gram = tuple(tokens[i:i+n])
            gram_str = ' '.join(gram)
            gram_hash = hashlib.sha256(gram_str.encode()).hexdigest()[:16]
            
            if gram_hash in fingerprint_db:
                match_info = fingerprint_db[gram_hash]
                matches.append({
                    'match_hash': gram_hash,
                    'matched_ngram': gram_str,
                    'n': n,
                    'benchmark_source': match_info['source_file'],
                    'training_file': str(filepath.name),
                    'token_position': i,
                    'context': ' '.join(tokens[max(0, i-5):i+n+5])
                })
    
    return matches


def scan_corpus(fingerprint_path: str, corpus_dir: str, report_path: str):
    """Scan entire training corpus for benchmark contamination."""
    
    # Load fingerprints
    with open(fingerprint_path) as f:
        fp_data = json.load(f)
    fingerprint_db = fp_data['fingerprints']
    print(f"Loaded {len(fingerprint_db)} fingerprints from {fp_data['source_files']} benchmark files.")
    
    corpus_path = Path(corpus_dir)
    all_matches = []
    files_scanned = 0
    contaminated_files = set()
    
    # Scan all text files in corpus
    for ext in ('*.txt', '*.md', '*.json', '*.jsonl'):
        for filepath in corpus_path.rglob(ext):
            matches = scan_file(filepath, fingerprint_db)
            if matches and 'error' not in matches[0]:
                all_matches.extend(matches)
                contaminated_files.add(str(filepath))
            files_scanned += 1
    
    # Write report
    with open(report_path, 'w') as f:
        f.write("=" * 70 + "\n")
        f.write("IUK BENCHMARK CONTAMINATION SCAN REPORT\n")
        f.write("=" * 70 + "\n\n")
        
        f.write(f"Files scanned:           {files_scanned}\n")
        f.write(f"Contaminated files:      {len(contaminated_files)}\n")
        f.write(f"Total fingerprint hits:  {len(all_matches)}\n\n")
        
        if not all_matches:
            f.write("STATUS: CLEAN — No benchmark content detected in training corpus.\n")
            f.write("Training run may proceed.\n")
        else:
            f.write("STATUS: *** CONTAMINATION DETECTED — DO NOT PROCEED WITH TRAINING ***\n\n")
            f.write("Contaminated files:\n")
            for cf in sorted(contaminated_files):
                f.write(f"  - {cf}\n")
            f.write("\nDetailed matches:\n")
            f.write("-" * 70 + "\n")
            for match in all_matches[:100]:  # Cap output at 100 matches
                f.write(f"File:      {match['training_file']}\n")
                f.write(f"Source:    {match['benchmark_source']}\n")
                f.write(f"N-gram:    {match['matched_ngram']}\n")
                f.write(f"Context:   ...{match['context']}...\n\n")
        
    print(f"\nScan complete. Report written to: {report_path}")
    print(f"STATUS: {'CLEAN' if not all_matches else 'CONTAMINATED — TRAINING BLOCKED'}")
    return len(all_matches) == 0


if __name__ == '__main__':
    import argparse
    parser = argparse.ArgumentParser()
    parser.add_argument('--fingerprints', required=True)
    parser.add_argument('--corpus', required=True)
    parser.add_argument('--report', default='iuk_contamination_report.txt')
    args = parser.parse_args()
    
    is_clean = scan_corpus(args.fingerprints, args.corpus, args.report)
    exit(0 if is_clean else 1)  # Non-zero exit blocks automated pipelines
```

## 5. Training Pipeline Integration

### 5.1 Required Gate: Scan Before Train

Insert the contamination scan as a **hard gate** in your training pipeline. Training must not proceed if the scan exits non-zero.

**Shell integration:**
```bash
#!/bin/bash
# iuk_training_gate.sh — Run before fine-tuning

FINGERPRINTS="/home/ryan/iuk_benchmark/fingerprints/iuk_fingerprints_latest.json"
CORPUS_DIR="/home/ryan/training_data/"
REPORT="iuk_contamination_$(date +%Y%m%d_%H%M%S).txt"

echo "=== IUK Air-Gap Check ==="
python iuk_scan.py \
    --fingerprints "$FINGERPRINTS" \
    --corpus "$CORPUS_DIR" \
    --report "$REPORT"

SCAN_EXIT=$?

if [ $SCAN_EXIT -ne 0 ]; then
    echo ""
    echo "*** TRAINING BLOCKED: Benchmark contamination detected. ***"
    echo "*** Review: $REPORT ***"
    exit 1
fi

echo "Air-gap verified — CLEAN. Proceeding with training."
# Continue to fine-tuning command here
```

## 6. Post-Training Verification — Canary Probe Test

After fine-tuning completes, run a secondary verification using **canary probes** — questions from the benchmark that are verbatim fed to the newly fine-tuned model to check for rote memorization.

### 6.1 Canary Probe Selection

Select 20 questions from across the benchmark as canary probes. Criteria:
- 4 from T1 (1 per type category)
- 4 from T2
- 4 from T3
- 4 from T4 (one per sub-type)
- 4 from T5

These 20 questions are the ONLY benchmark questions shown to the model in any context during this test. All other 1,273 questions remain unseen.

### 6.2 Canary Probe Script

```python
#!/usr/bin/env python3
"""
IUK Canary Probe — Post-Training Memorization Check
Tests whether fine-tuned model has memorized benchmark answer keys verbatim.
"""

import json
import re


CANARY_QUESTIONS = [
    # Format: (question_id, question_text, key_phrases_from_answer_key)
    # Populate with 20 selected benchmark questions
    # key_phrases = 3-5 specific phrases that appear verbatim in the answer key
    # A clean model that REASONS correctly will produce these phrases naturally
    # A memorizing model will produce them too — but the WRONG ANSWER DETECTION is the tell
    {
        'id': 'IUK-T4-ADVC-XXX',
        'question': '[question text here]',
        'key_phrases': ['phrase 1 from answer key', 'phrase 2'],
        'wrong_answer_phrase': '[specific wrong answer phrase that only appears in the answer key as a wrong answer]'
    },
]


def probe_model(model_response: str, canary: dict) -> dict:
    """
    Analyze model response for signs of memorization.
    
    A memorizing model may reproduce answer key phrasing verbatim.
    A reasoning model will produce correct answers using its own language.
    
    Key detection: Wrong answer phrases appearing in model output = 
    the model has seen the answer key (why would it repeat what's wrong?)
    """
    response_lower = model_response.lower()
    
    result = {
        'id': canary['id'],
        'key_phrases_found': [],
        'wrong_answer_phrases_found': [],
        'memorization_risk': 'LOW'
    }
    
    for phrase in canary['key_phrases']:
        if phrase.lower() in response_lower:
            result['key_phrases_found'].append(phrase)
    
    # CRITICAL: Wrong answer phrases in output = almost certain memorization
    if canary.get('wrong_answer_phrase'):
        if canary['wrong_answer_phrase'].lower() in response_lower:
            result['wrong_answer_phrases_found'].append(canary['wrong_answer_phrase'])
            result['memorization_risk'] = 'HIGH — WRONG ANSWER TEXT REPRODUCED'
    
    # Heuristic: if all key phrases are reproduced verbatim, flag for review
    if len(result['key_phrases_found']) == len(canary['key_phrases']):
        result['memorization_risk'] = 'ELEVATED — All key phrases reproduced; review for rote vs. reasoning'
    
    return result
```

### 6.3 Memorization vs. Reasoning: The Distinction

The canary probe is NOT a pass/fail test on correctness alone. A well-trained model SHOULD answer these correctly — that's the point of fine-tuning.

**The contamination signal is:**
1. **Wrong answer phrases appearing in the response** — the model reproduces text from the answer key section labeled "COMMON WRONG ANSWERS." A model that has never seen the answer key has no reason to reproduce these specific wrong-answer phrasings.
2. **Verbatim reproduction of answer key text** — if the model reproduces 3+ consecutive sentences from the answer key word-for-word, it has likely seen the answer key during training.
3. **Correct answer with wrong reasoning** — the model produces the correct final answer through reasoning that was explicitly marked incorrect in the answer key. This is the hardest contamination signal to detect and requires human review.

**Not a contamination signal:**
- Model produces correct answers (expected if training worked)
- Model uses similar technical terminology (shared domain vocabulary)
- Model echoes the question's framing (normal behavior)

## 8. What This Protocol Does NOT Cover

- **Benchmark question generation** — this document assumes the benchmark is already generated and partitioned. See benchmark files.
- **Model evaluation scoring** — see the IUK Scoring Judge system prompt.
- **Semantic similarity matching** — this protocol uses exact n-gram matching. A paraphrased version of a benchmark question would not be caught. However, paraphrasing during training would also substantially change the question's adversarial character, reducing the risk to benchmark validity.
- **Internet pre-training contamination** — if the base model (Mistral 7B, Llama 3.1 8B) was pre-trained on data that included these benchmark files (extremely unlikely but theoretically possible for publicly posted content), this protocol will not detect it. Keep benchmark files off public repositories.
